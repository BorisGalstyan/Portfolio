# Telegram FAQ Bot (RU)

Простой Telegram-бот-визитка с кнопочным меню и разделом FAQ. Подходит для малого бизнеса, личных проектов и каналов: бот отвечает на частые вопросы, показывает услуги/тарифы и контакты.

## Возможности

- Команда `/start` с приветствием и главным меню.
- Разделы: «О компании», «Услуги и тарифы», «FAQ», «Контакты».
- Кнопки Inline-клавиатуры, навигация без ввода команд.
- Лёгкая адаптация текстов под любой проект.

## Запуск

1. Создайте бота через BotFather и получите токен.
2. Добавьте токен в переменную `BOT_TOKEN_FAQ` (в `.env` или прямо в код).
3. Установите зависимости: `pip install -r requirements.txt`.
4. Запустите
